RCx Reporter

This is a simple tool created with the intention of benefiting the retrocommissioning (RCx) industry. With better data analysis we can provide better results which will only help promote more work in the existing building commissioning industry. With more work in the industry we can lower green house gas emmissions, save energy, lower utility costs, and create better indoor environments.
